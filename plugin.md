#### Как написать плагин

##### Самовызывающиеся плагины [ autorun ]
```javascript
  vnjs.on(function(){
    console.log('autorun');
  });

```
##### Вызываются из пользовательского скрипта
```javascript
  vnjs.on('alert', function(param, ctx){
    alert(ctx.scene)
  })

```
##### Файловая структура плагина

```plane_text
plugins
├── alert
│   ├── index.js
│   ├── package.json
│   └── README.md
├── debug
│   ├── index.js
│   ├── package.json
│   └── README.md

```
